# Predicting S&P 500 Stock Prices

## Overview
This project focuses on analyzing and predicting stock price movements of the **S&P 500 Index (`^GSPC`)** using historical data.  
By leveraging Python libraries and data from Yahoo Finance, the notebook explores long-term market trends, visualizes patterns, and applies predictive models to forecast future prices.

---

## Features
- Retrieves complete S&P 500 historical data using `yfinance`
- Cleans and processes time-series data
- Visualizes market trends, moving averages, and volatility
- Implements predictive modeling for price forecasting
- Written entirely in Jupyter Notebook for transparency and reproducibility

---

## Tools & Libraries
- **Data Source:** `yfinance`
- **Data Analysis:** `pandas`
- **Modeling:** `scikit-learn`
- **Environment:** Jupyter Notebook (Python 3.x)
